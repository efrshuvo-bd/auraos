# Documented serial transcript for AuraOS kernel bring-up on QEMU virt.
# `scripts/run-qemu.ps1` uses `-machine virt,gic-version=2`, raw `-kernel build/aura-kernel.bin`,
# `-initrd build/initrd.cpio`, and muxes PL011 UART + VirtIO console onto stdio
# (order may interleave slightly).
#
# Kernel early boot / sched lines → UART
# Guest init/agent/shell writes     → VirtIO console TX (SYS_WRITE)
# Guest SYS_READ                   → VirtIO console RX (polled; empty unless typed)
#
# Init yields until Agent Core posts READY; shell then requests help + system_status
# over IPC. Shell asks agent to shutdown so the scheduler can reach idle.
# During shell's busy-wait, CNTP may preempt several times; up to 3
# `sched: preempt pid=…` lines are logged (further preemption is silent).
