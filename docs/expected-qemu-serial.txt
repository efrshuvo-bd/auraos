# Documented serial transcript for AuraOS kernel bring-up on QEMU virt.
# `scripts/run-qemu.ps1` muxes PL011 UART + VirtIO console onto stdio, so you
# see both streams in one terminal (order may interleave slightly).
#
# Kernel early boot / sched lines → UART
# Guest init/agent/shell writes     → VirtIO console TX (SYS_WRITE)

virtio: console TX ready
phase2: timer tick armed
syscall: table ready (write/yield/exit/ipc)
phase2: scheduler + syscalls + ipc ready

phase2/3: userspace init scheduled
AuraOS entering scheduler

init: starting Agent Core (required)
init: starting shell
